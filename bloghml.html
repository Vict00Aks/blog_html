O que é HTML

HTML (Linguagem de Marcação de Hipertexto) é a espinha dorsal da web. Ele define a estrutura básica dos documentos da web usando uma variedade de tags e atributos.

É uma linguagem de marcação que utiliza HTML tags para descrever elementos como títulos, parágrafos, listas, links, imagens e muito mais.

Nota: Este é apenas um pequeno resumo.

Tags principais

As principais tags em HTML incluem:

  • <h1> até <h6>: Define cabeçalhos de diferentes níveis.
  • <p>: Define um parágrafo.
  • <mark>: Destaca o texto.
  • <small>: Define texto pequeno.
  • <i>: Define texto em itálico.
  • <u>: Define texto sublinhado.
  • <strong>: Define texto forte, importante.
  • <ol>, <ul>, <li>: Criam listas ordenadas e não ordenadas.
  • <a>: Define um link.
  • <hr>: Cria uma linha horizontal.
  • <sub>: Define texto subscrito.
  • <sup>: Define texto sobrescrito.
  • <blockquote>: Define uma citação em bloco.

IDs/Atributos

Em HTML, você pode usar atributos para fornecer informações adicionais sobre elementos. Por exemplo, o atributo id é usado para identificar um elemento específico.

O que faz target="_blank"

O atributo target="_blank" em um link faz com que o link seja aberto em uma nova janela ou guia do navegador.
